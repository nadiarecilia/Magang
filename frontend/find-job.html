Web Developer

Deskripsi Pekerjaan:

Bertanggung jawab dalam merancang, mengembangkan, dan memelihara aplikasi web secara end-to-end. Kandidat harus mampu mengimplementasikan desain UI ke dalam halaman web yang responsif serta mengelola logika backend menggunakan framework Laravel.

Persyaratan:

  • Pendidikan minimal S1 Teknik Informatika atau bidang terkait
  • Pengalaman minimal 2 tahun sebagai Web Developer
  • Menguasai HTML, CSS, JavaScript (preferably React atau Vue)
  • Menguasai PHP dan framework Laravel
  • Familiar dengan MySQL, Git, REST API, dan deployment ke server

Skill:

ReactJS Git VSCode
UI/UX Designer

Deskripsi Pekerjaan:

Posisi UI/UX Designer akan terlibat langsung dalam proses desain aplikasi web dan mobile mulai dari pembuatan wireframe, user flow, hingga prototipe. Kandidat akan dibimbing untuk memahami proses user research dan usability testing serta bekerja sama dengan developer dalam implementasi desain ke produk nyata.

Persyaratan:

  • Mahasiswa aktif jurusan Desain Komunikasi Visual atau Teknik Informatika
  • Memiliki portofolio desain digital (Figma, Adobe XD)
  • Mampu berpikir kritis dan memiliki sensitivitas terhadap user experience
  • Bersedia magang minimal 3 bulan

Skill:

Figma Adobe XD Sketch

Deskripsi Pekerjaan:

  • Memiliki kemampuan analisis data dan komunikasi yang baik
  • Kreatif, inisiatif tinggi, dan mampu bekerja dalam tim

Google Analytics Google Ads SEMrush
